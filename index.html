Game Rules

1.The primary goal of the game is to accumulate the highest score possible within the time limit.

2.The game board consists of multiple bubbles, each labeled with a number from 1 to 9, where 9 is the maximum number that can appear in the hit section.

3.There is a designated hit section that displays a specific number. You must click on bubbles with the same number as shown in the hit section to score points..

4.When you correctly click on a bubble with the number matching the one in the hit section, you earn 10 points.

5.The game has a time limit of 60 seconds. You must pop as many matching bubbles as possible within this time frame.

6.A timer is displayed on the screen, counting down from 60 seconds to 0. When the timer reaches 0, the game ends.

7.The reset button can be used to restart the game at any time. Clicking the reset button will reset the score to zero and restart the timer at 60 seconds.

8.The game ends when the timer reaches 0. Your final score is displayed, and you have the option to restart the game using the reset button.
